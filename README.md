## 📝 Description

A **neobrutalist personal portfolio** built with pure HTML, CSS, and vanilla JavaScript. Features interactive particle constellation systems, a custom magnetic cursor, 3D tilt skill cards, text scramble reveal animations, glitch effects, and much more — all without any frameworks or libraries.

---

## ✨ Features

- 🖱️ **Custom cursor** with magnetic interaction on interactive elements
- 🌌 **Particle constellation** hero background with connected nodes
- 🃏 **3D tilt skill cards** that respond to mouse movement
- 🔤 **Text scramble reveal** animations on scroll
- ⚡ **Glitch effect** on name heading
- 📊 **Scroll progress indicator** bar at the top of the page
- 🔢 **Animated counter stats** that count up on scroll
- 📱 **Fully responsive design** across all devices
- ✉️ **Contact form** with `mailto` integration
- ⌨️ **Typing animation** in the hero subtitle

## 🚀 Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/portfolio.git
   cd portfolio
   ```

2. **Open in browser**

   Simply open `index.html` in your favourite browser — no build step required!

   ```bash
   # macOS
   open index.html

   # Windows
   start index.html

   # Linux
   xdg-open index.html
   ```

## 🌐 Deployment

### Netlify

1. Push your repository to GitHub.
2. Go to [Netlify](https://www.netlify.com/) → **Add new site** → **Import an existing project**.
3. Connect your GitHub repo.
4. Set **Publish directory** to `/` (root).
5. Click **Deploy site**.

### GitHub Pages

1. Go to your repository **Settings** → **Pages**.
2. Under **Source**, select the branch (e.g., `main`) and folder (`/ (root)`).
3. Click **Save** — your site will be live at `https://<username>.github.io/portfolio/`.
